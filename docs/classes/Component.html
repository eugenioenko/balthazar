Class Component

This is a base class of the component of the engine. The engine consist of multiple components which perform various tasks. Some Components form part of the core of the Engine, others could be added as need at runtime. When the Engine is ready, it will create the instance of the component and pass itself as the engine parameter. Each Component instance has access to the engine

The instance of the engine, this will be passed by the engine
